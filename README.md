## üéØ Key Topics Covered

- **Database Design**: Scalability, reliability, and maintainability principles
- **Storage Engines**: LSM Trees, B+ Trees, and indexing strategies
- **Data Processing**: OLTP vs OLAP, data warehousing
- **Data Encoding**: Serialization formats and schema evolution
- **Replication**: Single-leader, multi-leader, and leaderless approaches
- **Consistency**: Replication lag, quorum consensus, and consistency models
- **Partitioning**: Sharding strategies and consistent hashing
- **Transactions**: ACID properties, isolation levels, and concurrency control
- **Anomalies**: Lost updates, write skew, and serializability challenges
